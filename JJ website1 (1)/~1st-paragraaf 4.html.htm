De top 10 meest gebruikte programmeertalen zijn op dit moment: 

1.python: 

 Python is gemaakt door Guido van Rossum rond het begin jaren 90 (1991). Python is op dit moment de meest gebruikte programmeertaal in de wereld, en word tot de dag van vandaag nog steeds ge update 

2.c 

Een opvolgeling tor programmeertaal B, C is ontstaan in bell labs door ene richie dit gebeurde tussen de jaren 1972 en 1973 om bepaalde utiliteiten van unix te construeren. C is lanzamerhand een van de meest populaire programeertalen geworden 

3.java 

Java is ontstaan in 1995 gemaakt door james gosling met de opdracht "Maak iets cools". Java was origineel gebruikt voor settopboxen maar vanwege java's simpliciteit word het hedertijd gebruikt voor vele dingen zoals populaire hit game minecraft. 

4.c++ 

C++ is ontstaan in 1983, ontworpen door Bjarne Stroustrup in Denemarken. C++ was een verbetering van programmeertaal C. Programeertaal java is gebaseerd op c++. 

5.C#. 

C# (C sharp) is een multifunctionele programmeertaal, gemaakt in 2000 gemaakt door Andre Hejsberg. C# was gemaakt vanuit microsoft en werd gezien als een internationale Standaard 

6.Visualbasic 

Visualbasic is ontstaan in 1992, en gemaakt door microsoft. Visual basic heeft zich door de jaren heen steeds meer ontwikkeld er zijn 6 verschillende versies gemaakt als verschillende rendities van de programeertaal. Op dit moment zitten we op versie 6.0 

7.JavaScript 

Javascript is verschenen in 1995 en is gemaakt door brendan eich. Javascript is een van de kerntechnologieen van de wereldwijdeweb. Javascript is gemaakt als scripttaal om webpaginas interactief te maken 

8.assembly language 

Assembly language is 75 jaar geleden gemaakt in 1947. Assembly language is een collectie van low level programming. De bedoeling van assembly language is meestal om machines te gebruiken 

9.SQl 

SQL (Structured query language) ook wel PL (Procedural language) genoem is ontstaan in 1992. Net zoals visualbasic zijn er meerdere rendities op dit moment zit sql op 8.0. 

10.swift 

Ontwikkeld in 2014 en gemaakt door apple, swift is een van de nieuwere en gevorderde programmeertalen. Sinds swift ook gemaakt is door apple is het ontwikkeld als besturing voor ios
